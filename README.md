# 🚀 响应式数据驱动与工程化实践项目

这是一个用于展示 Vue3 组件解耦、环境变量管理及自动化部署的演示项目。

## 🛠 技术栈

- 框架：Vue 3 (Composition API)
- 脚本：TypeScript
- 构建：Vite
- 部署：Vercel (CI/CD)

## ✨ 核心特性

1. **数据与视图解耦**：基于 Schema 驱动的动态表单设计。
2. **多环境管理**：通过 `.env` 实现开发与生产环境的配置分离。
3. **自动化流水线**：接入 GitHub Actions，实现 push 即部署。

## 运行项目

1. `npm install`
2. `npm run dev`
